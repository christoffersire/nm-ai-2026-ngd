# NM i AI 2026 — NorgesGruppen Object Detection

Grocery product detection and classification on store shelf images.

## Scoring

Score = 0.7 × detection_mAP@0.5 + 0.3 × classification_mAP@0.5

- Detection component: IoU ≥ 0.5, category ignored (max 70%)
- Classification component: IoU ≥ 0.5 + correct category_id (remaining 30%)

## Data

- 248 shelf images, 22,731 annotations, 356 categories (nc=356, ids 0-355)
- 41 categories have only 1 annotation; category 355 = `unknown_product`
- Training data: `~/Downloads/train/` (images/ + annotations.json)
- Product reference images: `~/Downloads/NM_NGD_product_images/`

## Project Structure

```
data/
  prepare_yolo.py        # COCO → YOLO format (full-class + single-class)
  audit.py               # Data audit + category mapping
  val_split.json         # Fixed train/val split (199/49)
  category_mapping.json  # Category ID → name mapping
scoring/
  scorer.py              # Local hybrid scorer (mirrors competition logic)
  tests/test_scorer.py
train/
  train_detector.py      # YOLOv8 training script
submission/
  run.py                 # Competition entry point
  validate_submission.py # Zip structure checker
datasets/                # Generated by prepare_yolo.py
  full-class/            # All 356 categories
  single-class/          # All products → class 0
```

## Workflow

### 1. Data Preparation

```bash
python data/prepare_yolo.py
```

Converts COCO annotations to YOLO format. Creates `datasets/full-class/` and `datasets/single-class/` with train/val split (199/49 images, stratified to keep rare categories in train).

### 2. Training (on GCP VM)

```bash
gcloud compute ssh yolo-trainer --zone=europe-west1-c --project=ai-nm26osl-1799

cd ~/nm-ai-2026-ngd
python3 train/train_detector.py \
  --dataset datasets/full-class/data.yaml \
  --model yolov8m.pt \
  --epochs 100 --imgsz 1280 --batch -1 --patience 20 \
  --name full-class-v1 --device 0
```

VM: g2-standard-8, 1x NVIDIA L4 (24GB), Python 3.10, PyTorch 2.7.1, ultralytics 8.4.24.

### 3. Local Evaluation

```bash
python scoring/scorer.py \
  --predictions predictions.json \
  --ground-truth ~/Downloads/train/annotations.json \
  --val-split data/val_split.json
```

### 4. Submission

```bash
python run.py --input /data/images --output /output/predictions.json
```

Package as ZIP with `run.py` + model weights at root. Max 420 MB, ≤3 weight files, ≤10 .py files.

**Competition sandbox**: Python 3.11, PyTorch 2.6.0+cu124, ultralytics 8.1.0, NVIDIA L4, 300s timeout, no internet.

### Submission Limits

- 3 submissions/day, 2 in-flight max
- Resets at midnight UTC
